# Personal Finance Dashboard

## Description
A minimalist web application for **personal finance management and analysis**, developed during the summer internship at **ASEM**. The project addresses the issue of monitoring expenses by providing an intuitive interface for recording transactions and visualising the budget graphically.

## Key Features
* **Transaction Recording**: Adding money flows (Income/Expenses) with details about amount, category, and date.
* **Graphical Visualization**: Automatic generation of charts via **Chart.js** to see the distribution of expenses by category.
* **Budget Management**: Setting a monthly limit and real-time monitoring of the financial balance.
* **Local Storage**: Using **localStorage** to keep user data directly in the browser, without requiring a backend.
* **Clean Interface**: Adaptive and easy-to-use design.

## Technologies Used
* **Languages**: HTML5, CSS3, JavaScript (ES6+).
* **Libraries**: [Chart.js](https://www.chartjs.org/) for data visualization.
* **Versioning**: Git & GitHub.
* **Deployment**: [GitHub Pages](https://pages.github.com/).

## Installation and Usage
The project does not require a server or external databases.

1.  **Cloning the Repository**:
    ```bash
    git clone https://github.com/username/personal-finance-dashboard.git
    ```
2.  **Launch**:
    * Navigate to the project folder.
    * Open the `index.html` file using any web browser (Chrome, Edge, Firefox).
3.  **Live Usage**:
    * The application can be accessed directly via the link generated by GitHub Pages (e.g., `https://username.github.io/personal-finance-dashboard/`).
